# HolyTime-Mineflayer
Апгрейд предыдущего бота под фт, чтобы решал капчи на хт
# Установка
* Открыть install_depends.bat
* Или же самому прописать в директории с ботом:
- npm i flayercaptcha
- npm i node-fetch@2
- npm i Sharp
# Примечание
И естественно, если не установлен mineflayer то npm install mineflayer
